# Telco Customer Churn Analysis – Python & Power BI

## Descripción
Este proyecto analiza los datos de clientes de una compañía de telecomunicaciones para identificar patrones de **churn** (clientes que abandonan el servicio). El objetivo es extraer insights que permitan entender el comportamiento de los clientes y mejorar la retención.

## Herramientas utilizadas
- **Python**: limpieza y preparación del dataset (`Telco_Customer_Churn_clean.csv`) usando pandas y numpy.
- **Power BI**: creación de dashboard interactivo con gráficos de columnas apiladas para visualizar churn por tipo de contrato.

## Dataset
- El dataset original se descargó de [Kaggle](#) y se limpió para este proyecto.
- Archivo incluido: `Telco_Customer_Churn_clean.csv`.

## Dashboard
- Gráfico principal: porcentaje de clientes que abandonan (Churn Yes/No) según tipo de contrato.
- Colores consistentes y etiquetas de datos para mejorar la comprensión de los insights.
